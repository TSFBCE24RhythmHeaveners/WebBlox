# WebBlox
Welcome to Web Blox, the program that lets you build perfect HTML, PHP, ASP, XML, RSS, Pearl, JS, CGI, CSS and many other web releasted files! Web Blox is a way of making these files in a much more visual way than in notepad. You may argue that something like Dream weaver may be better but those kind of HTML editors make you learning HTML extremely hard but Web Blox helps you learn HTML with it's help screen for every element and does it in a much more friendlier interface. Enjoy using Web Blox!
